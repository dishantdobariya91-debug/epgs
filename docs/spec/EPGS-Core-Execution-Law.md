\# EPGS Core Execution Law



\## Permission States

\- ALLOW

\- ASSIST

\- BLOCK



\## Stop Dominance

If STOP is issued or permission is BLOCK, execution MUST terminate.



\## Final Execution States

\- EXECUTED

\- TERMINATED



\## Fail-Closed Rule

Missing or invalid inputs MUST result in termination.



No other final states are permitted.
